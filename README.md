### 복합예술 추천 시스템 
> 맞춤형 뮤지컬 및 연극 추천 추천 서비스

## Usage
### Install virtual environment

```bash
python (py) -m venv venv
source venv/Scripts/activate
```

### Run
```bash
python manage.py runserver
```
